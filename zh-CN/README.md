# PHPOpenCV

## 目的
PHP在Web开发领域已经可以达到登峰极造的地步，但在人工智能方面和python，C++等语言相对比，缺少很多可用的库，
所以基于C\C++接口实现PHP版的OpenCV，以丰富php这方面的领域。
